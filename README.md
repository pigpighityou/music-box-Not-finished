# 16music-box(请注意：半成品！)

## 用vscode等打开

## 终端安装包管理器npm||yarn（版本要新）

```markdown
npm install
```

## 终端安装数据服务器

```markdown
    $ git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git
    $ cd NeteaseCloudMusicApi
    $ npm install
```

## 运行服务器

```markdown
    $ node app.js
```
